A lite template for building an Electron app with React 18 + TypeScript 4.8 and Webpack 5

### Quick start

- Clone

```bash
  git clone https://github.com/luckrnx09/electron-react-ts-starter
```

- Install the dependencies

```bash
  npm install
```

- Run for development

```bash
  npm run dev
```

- Publish for production

```bash
  npm run build
  npm run package
```
